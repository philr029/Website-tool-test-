Features

Everything you need to
trust your forms.

No custom code. No flaky scripts. Just a JSON config and a single command.

Config-driven testing
Define your entire test suite in JSON or YAML. No TypeScript, no Playwright boilerplate — just describe what to fill and click.
Lightning fast
Headless Chromium runs every form test in under 3 seconds. Run your full suite in CI in seconds, not minutes.
CAPTCHA detection
Automatically detects CAPTCHAs and reCAPTCHAs, flags them for manual review instead of silently failing.
Live dashboard
Beautiful HTML dashboard with pass/fail stats, per-site breakdowns, and a progress bar. Load any JSON report instantly.
GitHub Actions ready
Drop-in workflow file included. Push your config, get automated test results on every commit — zero setup required.
Multi-step form support
Test complex multi-page flows, wizard steps, and dynamic forms. Conditional logic handled automatically.
How it works

Up and running
in 3 steps.

From zero to automated form testing in under 5 minutes.

1
Write a config
Define your target URL, selectors, and input values in a simple JSON file. No code required.
2
Run the tester
A single command launches headless Chromium, fills every form, and records results to a JSON report.
3
Review results
Open the dashboard, load your report, and instantly see what passed, failed, or needs manual review.
